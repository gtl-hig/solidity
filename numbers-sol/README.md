# numbers-sol (hardhat) Project

This project demonstrates a basic Hardhat usage and simple Solidity constructs.
It comes with a sample contract, a test for that contract, and a simple frontend module
for interacting with the contract through web3 and ethers.js Javascript libraries.

# Deployment

This contract has been deployed on Sepolia at address:
`0x32c33a8ea5292c224A27A784dD47cc2b512AD9AB`

# Prerequisites

You will need to have Node.js and npm installed on your machine. You can download them from [here](https://nodejs.org/).
Then do:

```shell
cd numbers-sol
npm install
```

See `numbers` for deployment options.

# Extras

Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
```
